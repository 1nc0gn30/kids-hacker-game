# KidGame3

Section/scene-structured Python + Pygame story game.

Current build includes:
- Section 1
- Scene 1 (`Parrot OS desktop -> email alert -> opened warning email -> malware popup`)
- Scene 2 (`loopback packet capture review`)

## Run

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python main.py
```

## Controls

- Press letter keys `a-z` to advance the scene.
- Every 30 letter key presses triggers the next beat.
- Press `Esc` to quit.
